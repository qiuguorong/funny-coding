## 重排（reflow）
### 怎么引发重排
1、页面渲染初始化
2、浏览器窗口尺寸改变
3、dom树结构变化（添加、删除dom）
4、dom位置改变
5、元素的盒模型改变（外边距、内边距、边框厚度、宽高等）
6、获取某些属性offsetTop、offsetLeft、 offsetWidth、offsetHeight、scrollTop、scrollLeft、scrollWidth、scrollHeight、clientTop、clientLeft、clientWidth、clientHeight、getComputedStyle
